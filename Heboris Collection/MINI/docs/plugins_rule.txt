-------------------------------------------------
付属のルールプラグインの説明
-------------------------------------------------

【プラグインファイル一覧】
plugin_basic/ars.c … ARS
plugin_basic/srs.c … SRS
plugin_basic/drs.c … DRS

【各ルールの説明】
ARS
	原作のヘボリスとほぼ同じ回転法則を持ったルールです。

SRS
	強引な壁登りや回転入れが可能な、初心者にも扱いやすいルールです。

DRS
	特異な回転法則や壁蹴りが特徴のルールです。使いこなすにはある程度の慣れが必要となります。

【ルール設定項目の説明】
EXT-ROTATE
	ONにすると、I字ブロックが壁蹴りができるようになります。（何回でも可能）
	また、I字ブロックとT字ブロックが1回だけ床を蹴ることができるようになります。
	SRSにはこの項目はありません。

LOCK RESET
	ONにすると、ブロックが接地した状態で左右に移動したり、回転したりすると
	ブロックが固定されるまでの時間が延長されるようになります。

ROTATE LIMIT
	LOCK RESETをONにしているとき、何回ブロックを回すと強制的に固定されるかを変更できます。
	INFINITEにすると回転を続ければいつまでもブロックが固定されません。
	LOCK RESETがOFFの場合は無視されます。

MOVE LIMIT
	LOCK RESETをONにしているとき、何回ブロックを移動すると強制的に固定されるかを変更できます。
	INFINITEにすると移動を続ければいつまでもブロックが固定されません。
	LOCK RESETがOFFの場合は無視されます。

BLOCK GRAPHICS
	表示されるブロックの絵柄を変更することができます。

ARE
	ブロックを固定させたあとに待ち時間があるかどうかを変更できます。
	OFFにすると待ち時間が無くなり、ブロックを置くとすぐに次のブロックが出現します。
	ON (ENABLE SKIP)にした場合は、待ち時間中に何かボタンを押すと待ち時間をキャンセルしてすぐに次のブロックが出現します。

LOCK BY UP KEY
	ハードドロップを使用したときにすぐにブロックが固定されるかどうかを変更できます。

LOCK BY DOWN KEY
	ブロックが接地しているときにソフトドロップを使用した場合、どれぐらい早くブロックが固定されるか変更できます。
	OFF：特に何も起こらない
	ON：即固定
	SLOW LOCK：微妙に早く固定が進む
	FAST LOCK：かなり早く固定が進む
	1G OR FASTER：ブロックの自然落下速度が速い場合にのみ即固定

ROTATE REVERSE
	ONにするとブロックが回転する方向が逆転します。
	OFFの場合 … A:左回転 B:右回転 C:左回転
	ONの場合  … A:右回転 B:左回転 C:右回転

SOFT DROP SPEED
	空中でソフトドロップを使用した場合の落下速度を変更できます。

MAX YOKOTAME
	左か右を押しっぱなしにした場合の横溜めの最大値を変更できます。
	値が小さいと序盤でもブロックがツルツル滑るようになります。
	INFINITEにすると横溜めの上限はありません。

INIT LR MOVE
	ONにすると、ブロックが出現するときに左か右を押しっぱなしにしていれば
	あらかじめブロックの位置がずれた状態で出現するようになります。

[] BACKGROUND
	[]の形のブロックが登場するモードで、背景を真っ白または真っ黒にすることができます。
	うまく使えばブロックが見やすくなります。
	OFF：背景を変更しない
	BLACK：真っ黒にする
	WHITE：真っ白にする

I BLOCK COLOR
L BLOCK COLOR
O BLOCK COLOR
Z BLOCK COLOR
T BLOCK COLOR
J BLOCK COLOR
S BLOCK COLOR
	登場するブロックの色を変更できます。

INITIAL ROTATE
	ONにすると、ブロックが出現するときにあらかじめ回転ボタンを押しておけば
	あらかじめブロックが回った状態で出現するようになります。

INITIAL HOLD
	ONにすると、ブロックが出現するときにあらかじめDボタンを押しておけば
	ブロックが出現すると同時にHOLD機能を使うことができます。

ENABLE WALLKICK
	ブロックが壁を蹴るかどうか変更できます。OFFにすると蹴らなくなります。

SELF LOCK SOUND
	ブロックを自分で固定させたときにse/self_lock.wavを鳴らすかどうか変更できます。

LOCK FLASH
	ブロックが固定したときに何フレーム間ブロックが白く光るかを変更できます。

[SAVE PRESET]
	5つあるプリセットのうちの1つに現在の設定を保存できます。
	保存したプリセットは下のLOAD PRESETから読み込めます。

[LOAD PRESET]
	上のSAVE PRESETで保存したプリセットを読み込みます。
