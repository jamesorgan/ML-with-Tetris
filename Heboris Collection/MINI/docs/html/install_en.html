Install
Install from scratch
1. Download "yaneuraoGameScript2000version1.64c" (ygs2k164c.lzh) from http://yaneurao.hp.infoseek.co.jp/ygs2k/
2. Unpack "YGS2000.exe" from the lzh file, then copy it into Heboris's folder.
3. Rename "YGS2000.exe" into "HEBORIS.EXE".
4. Run "HEBORIS.EXE".

Upgrade from previous version
Simply copy all files into your Heboris's folder.
If the superscription confirmation appears, please click "Yes" for all files.

Warning for Windows XP or newer Windows user
Please set compatibility mode to "Windows 2000" if you are using Windows XP, Vista or newer version.
http://www.microsoft.com/windowsxp/using/helpandsupport/learnmore/appcompat.mspx

How to change settings
Almost all settings are in CONFIG Screen. Usually you don't have to edit "heboris.ini".
