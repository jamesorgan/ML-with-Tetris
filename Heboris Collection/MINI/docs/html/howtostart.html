ゲームの始め方
タイトル画面で「PUSH START BUTTON!」と表示されているときにAボタンを押すと、このようなメニューが出ます。
ゲームを始めるときは「GAME START」を選んでAボタンを押してください。ルールセレクト画面に移動します。
「REPLAY」を選ぶと保存したリプレイを再生する画面へ、「CONFIG」を選ぶと設定画面に行きます。
使用する「ルール」を選ぶ画面です。
どのルールを使用するかによって、ブロックの回り方や操作性が変わります。
好きなものを選んでAボタンを押してください。
Bボタンを押すとキャンセルしてタイトル画面に戻ります。
ルールの詳細設定画面です。
操作性に関するいろいろ設定をここで変更することができます。
設定する内容は使用するルールによって変わります。
Aボタンを押すとモードセレクト画面に移動します。（[ ]で囲まれている項目を選んでいる場合を除く）
Bボタンを押すとルールセレクト画面に戻ります。
遊びたい「モード」を選ぶ画面です。
遊びたいモードを選択してAボタンを押してください。
Bボタンを押すとルール設定画面に戻ります。
モードの詳細設定画面です。
ゲームの難易度の設定や、ブロックの着地予定地表示（ゴーストブロック）の有無の切り替えなどを行います。
Aボタンを押すとゲームが始まります。
Bボタンを押すとモードセレクト画面に戻ります。
一部のモードではEボタンを押すとランキングを見ることができます。
