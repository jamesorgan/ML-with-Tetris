Add or remove plugins


(NOTE:The screen shot is Japanese version)
You can add or remove the plugins by using heboplugin.exe.
To add a plugin, drag a plugin file into the window of heboplugin.exe, or click "Add..." button and select the file.

Add...
Add single or multiple plugins to the list. You can select multiple files by pushing Ctrl key.

Remove
Remove the selected plugin from the list. This action does not erase the plugin file itself.

Up
Move the selected plugin up.

Down
Move the selected plugin down.

OK
Save the changes and quit the program.

Cancel
Discard the changes and quit the program.
