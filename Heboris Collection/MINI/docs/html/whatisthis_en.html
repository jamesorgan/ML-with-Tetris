What is this?
"HEBORIS U.E. MINI" is an unofficial patch for Hosiken's Tetromino based puzzle game, "Heboris".

Official page of original Heboris (Died)
http://www.hosiken.com/software/windows/heboris2001/

In this unofficial version, it has plugin support. You can add or remove functions from the game.
All game modes and rotation rules are provided as plugins.
To add or remove plugins, use "heboplugin.exe".

The title's "U.E." means "Unofficial Expansion".
