How to play

A small shape made up of four blocks (called Tetromino) will fall from top of the playfield.
The main object of this game is manipulate Tetromino and create the horizontal line of blocks without gaps.
You can move Tetromino to sideways, rotate by 90 or 180 degrees, or drop faster.

If the Tetromino lands to floor or blocks, it hardens (no longer manipulated by player). Then the next Tetromino will appear.

If you create the horizontal line of blocks without gaps, it will vanish from playfield.

When the block reaches to top of the playfield, your game ends.

You can reserve the current Tetromino for later use. (called Hold)
Press D button to move current Tetromino into "Hold Piece Area" (left of the piece preview).
You can press D button again to use the reserved Tetromino.
However, this feature cannot be used again until the reserved Tetromino locks.
