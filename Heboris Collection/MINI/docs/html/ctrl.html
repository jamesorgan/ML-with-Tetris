操作方法
ボタン名役割
UPハードドロップ（即落下）
カーソルを上に移動
DOWNソフトドロップ（高速落下）
カーソルを下に移動
LEFTブロックを左に移動
選択している項目の値を減らす
RIGHTブロックを右に移動
選択している項目の値を増やす
A回転
決定
B逆回転
キャンセル
C回転（Aと同じ）
これを押しながらLEFTまたはRIGHTボタンを押すと素早く値の変更が可能
Dホールド
E180度回転
Fモードによって役割が異なるボタン

このゲームではこれら10個のボタンを使用します。
どのボタンがキーボードのどのキーに対応するかは初回起動時に設定します。
キーボードのボタン割り当ては後からでも変更が可能です。
ゲームパッドやジョイスティックにも一応対応しています。

その他のボタン（ゲーム中の特殊操作を参照）
ボタン名役割

GIVEUPゲームの強制終了（通称捨てゲーキー）
SCREEN SHOTssフォルダにスクリーンショットを保存
EXTRA KEYデバッグメニューに入る（heboris.iniを書き換えて有効にしている場合）
FPS +1FPS（ゲームスピード）の最大値を1つ増やす
FPS -1FPS（ゲームスピード）の最大値を1つ減らす
