ゲームの設定を変える
タイトルメニューで「CONFIG」を選ぶと左の画面が表示されます。
ここでは見た目や操作方法の設定ができます。
GENERAL SETTING
ここではゲームに関する設定を変更できます。

NEXT PATTERN
出現するブロックが偏らないように補正をかけることができます。
どんな方法で偏り補正を行うかをここで変更できます。
RANDOM
完全にランダム
HEBO○（○は1～7の数字）
本家ヘボリスと同じ方式の生成方法です。
MEMORY4 (4 TRY)
過去に出現したブロックを4つまで記憶して偏りを少なくします。過去に出現したブロックが出現した場合は、4回まで再試行します。
MEMORY4 (6 TRY)
より偏りが少なくなっています。過去に出現したブロックが出現した場合は、6回まで再試行します。
7 PIECE BAG
7手毎に7種類のブロックが偏り無く全て出現します。
HEBO7と良く似ていますが、同じブロックが2回連続で来ることがあります。
CUSTOM
heboris.iniで設定した特定のパターンでブロックが出現します。
I-CHEAT
ヘボリス棒だけ出現します。

8WAY INPUT
斜め入力を許可するかどうか選択します。
OFF 許可しない
ON 許可する

FIELD BRIGHT
フィールドの背景の明るさを選択します。値が大きいほど明るくなります。

NEXT DISPLAY
NEXTブロックの表示数を0～6の範囲で選択します。

TOP FRAME
画面上部、NEXTブロックが表示されるエリアの飾りの種類を選択します。
TYPE A～TYPE Dの4種類あります。

LINE ERASE EFFECT
ブロックを消したときのエフェクトの種類を選択します。
NONE エフェクトを表示しません
POP OUT ブロックが跳ねて落ちます
SHATTER ブロックが粉々になります
SIMPLE-1 ブロックが点滅します
SIMPLE-2 ブロックが暗くなっていきます

DROP LIMIT
ハードドロップやソフトドロップをボタン押しっぱなしにしても使用できるかどうかを選択します。
OFF 使用可能
20G ONLY 非20Gでは押しっぱなしにしても使用可能、20Gでは不可
ON 使用不可能

ROTATE SOUND
ブロックを回したときse\rotate.wavを再生するかどうかを選択します。
OFF 再生しない
ON 再生する

MOVE SOUND
ブロックを動かしたときse\move.wavを再生するかどうかを選択します。
OFF 再生しない
ON 再生する

SHOW CONTROL
画面左に現在の操作状況をレバーとボタンのグラフィックで表示するかどうかを選択します。
OFF 表示しない
ON 常に表示する
REPLAY ONLY リプレイの時だけ表示する

SPAWN Y TYPE
ブロックの出現位置を選択します。
UNDER THE FRAME 枠の下
ABOVE THE FRAME 枠の上

NUMBER FONT TYPE
タイムやスコアなどの数字フォントの種類をTYPE AとTYPE Bの2種類から選択します。

SE VOLUME
効果音の音量を-10000～0で選択します。
0にするとそのままの音量で再生され、-10000にすると無音になります。

BGM VOLUME
音楽の音量を-10000～0で選択します。
0にするとそのままの音量で再生され、-10000にすると無音になります。

BGM TYPE
音楽の種類を選択します。
OFF 音楽を再生しません
WAV 拡張子が.wavの音楽を再生します
MP3 (NO ID3 TAG) 拡張子が.mp3の音楽を再生します（※１）
MIDI 拡張子が.midの音楽を再生します（※２）

※１：MP3の再生は非公式に対応しています。MP3のコーデックの種類によっては再生できない場合があります。また、ID3タグがついているMP3は使用できません。
※２：BGMが切り替わる際にハードディスクへのアクセスが発生します。またBGMのフェードアウトは行われず、一瞬で止まります。

BGM FADE SPEED
BGMがフェードアウトする早さを選択します。
値が大きいほど早くフェードアウトします。
0にすると一瞬で止まります。

SHOW FPS
画面下に現在のFPSを表示するかどうかを選択します。
OFF 表示しない
ON 表示する
CAPTION タイトルバーに表示する

FPS
標準FPSを選択します。値が小さいほどゲームスピードが遅く、大きいほど早くなります。

BACKGROUND
背景を表示するかどうか選択します。
OFF 表示しない
ON 表示する

HOLD SOUND
ホールドを使用したときにse\hold.wavを再生するかどうかを選択します。
INITIAL ONLY 先行ホールドのみ再生する
ON 常に再生する
OFF 再生しない

LANGUAGE
言語を選択します。いくつかのプラグインの表示に影響を与えます。
JAPANESE 日本語
ENGLISH 英語

E BUTTON
Eボタンの役割を変更します。
180 ROTATE 180度回転
REVERSE ROTATE 逆回転（Bボタンと同じ）

ENABLE SYSTEM BGM
ONにすると、タイトル画面でbgm\title.wav(またはbgm\title.mp3)を、セレクト画面でbgm\select.wav(またはbgm\select.mp3)を再生するようにします。
ただし、起動が遅くなり、メモリも圧迫します。
OFFにすると、タイトル画面とセレクト画面のBGMは無くなります。

EFFECT SPEED
ブロックの消去エフェクトの表示速度を「AUTO」「×1～5」で選べます。
AUTOにすると落下スピードにあわせて自動的に1～3倍速で調節します。
×1～5にすると表示速度が固定になります。
表示速度が速いほど表示される時間は短くなり、処理も軽くなります。

BLOCK SHADOW
LEFTにするとブロックの左側、RIGHTにするとブロックの右側にブロックの影を表示します。
フィールド枠も立体的な物に変化します。

HOLD
OFFにするとホールド機能が使用できなくなります。
INPUT SETTING
キーボードやジョイスティックの設定を変更できます。

[1P KEYBOARD SETTING]
1P側のキー配置を設定します。

[2P KEYBOARD SETTING]
2P側のキー配置を設定します。

[1P JOYSTICK SETTING]
1P側のジョイスティックのボタン配置を設定します。

[2P JOYSTICK SETTING]
2P側のジョイスティックのボタン配置を設定します。

[JOYSTICK MAX BUTTONS]
認識するジョイスティックのボタン数の最大値を設定します。

[INPUT TEST]
キー入力のテストができます。（A+Bでメニューに戻る）
SCREEN SETTING
画面サイズなどの設定ができます。変更した設定は次回起動時から有効になります。

SIZE
画面サイズを選択します。

MEMORY
グラフィックをビデオメモリに配置するか、システムメモリに配置するかを選択します。

COLOR
フルスクリーン時の画面の色数を選択します。

USE HIRES
画面サイズが640x480の時に使用するグラフィックの種類を選択します。
OFF 通常のグラフィックの2倍拡大（ビデオカードの種類によっては、画面がぼやける可能性があります）
ON 高解像度グラフィック（graphicshiフォルダとbackhiフォルダから読み込みます。快適に動かすには高性能なPCが必要です）
MUSIC TEST
音楽のテストができます。
LEFT 音楽の番号-1
RIGHT 音楽の番号+1
A 再生
B メニューに戻る
C 即座に停止
D 一時停止／再開
E フェードアウトして停止
SE TEST
効果音のテストができます。
LEFT 効果音の番号-1
RIGHT 効果音の番号+1
A 再生
B メニューに戻る
C 停止
戻る
