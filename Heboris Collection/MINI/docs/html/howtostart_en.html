How to start game
When the title screen appears, press A button to enter the main menu.
To start game, select "GAME START" and press A button.
To watch the replay file, select "REPLAY" and press A button.
To change the settings, select "CONFIG" and press A button.
Select your favorite rotation rule, and press A button.
(Press B button to return to the title screen)
This is rule options screen. You can change many settings in this screen.
Press A button to proceed to mode selection screen. (Unless you are selecting inclosed item, such as "[SAVE PRESET]")
Press B button to return to rule selection screen.
This is mode selection screen.
Select your favorite mode, and press A button.
Press B button to return to the rule options screen.
This is mode options screen.
You can change some of settings. ex. "Ghost block" that displays where the piece would drop.
Press A button to start game. Good luck!
Press B button to return to the mode selection screen.
In some modes, you can see ranking by pressing E button.
