Change the settings
When "CONFIG" is chosen in the title menu, a left screen is displayed.
You can change various settings in this menu.
GENERAL SETTING
You can change game-related settings here.

NEXT PATTERN
Select how the Tetromino's order is generated.
RANDOM
Classic "rand() % 7" randomizer.
HEBOx (x=1,2,3,4,5,6,7)
Same randomizer as original Heboris.
MEMORY4 (4 TRY)
Tetromino that appeared in the past are memorized up to four. When the Tetromino that appeared in the past appears, it retries it up to four times.
MEMORY4 (6 TRY)
When the Tetromino that appeared in the past appears, it retries it up to six times.
7 PIECE BAG
Generates a sequence of all seven Tetrominos permuted randomly.
CUSTOM
Use fixed Tetromino pattern in heboris.ini.
I-CHEAT
Only the I-Tetromino appear.

8WAY INPUT
Enable or disable the diagonal moves.
OFF Disable
ON Enable

FIELD BRIGHT
Select translucent level of the playfield background.

NEXT DISPLAY
Select how many piece previews are visible. (from 0 to 6)

TOP FRAME
Select the decoration of piece preview area. There are four types. (from TYPE A to TYPE D)

LINE ERASE EFFECT
Select the animation of line-clear.
NONE No animation.
POP OUT The line of blocks pops out from the playfield.
SHATTER The blocks shatters and disappears.
SIMPLE-1 The blocks blinks and disappears.
SIMPLE-2 The blocks darkens and disappears.

DROP LIMIT
Select continuation soft/hard drop limitations.
OFF No limitations.
20G ONLY In 20G speed, player can't do continuity soft/hard drop. Otherwise, there are no limitations.
ON Player can't do continuity soft/hard drop.

ROTATE SOUND
Select whether to play se\rotate.wav when the Tetromino is turned.
OFF Disable
ON Enable

MOVE SOUND
Select whether to play se\move.wav when the Tetromino is moved.
OFF Disable
ON Enable

SHOW CONTROL
Select whether to display player's input to game screen.
OFF Disable
ON Enable
REPLAY ONLY Enable in replay mode, otherwise disable

SPAWN Y TYPE
Select Tetromino's spawn position.
UNDER THE FRAME Tetromino spawns inside of the playfield
ABOVE THE FRAME Tetromino spawns outside of the playfield

NUMBER FONT TYPE
Select the numeric font type from TYPE A or TYPE B.

SE VOLUME
Select the volume of sound effect from -10000 to 0.
-10000 means no sound, 0 means maximum volume.

BGM VOLUME
Select the volume of music from -10000 to 0.
-10000 means no sound, 0 means maximum volume.
NOTE: There is no effect if you are using MIDI music.

BGM TYPE
Select the music type. Enabling this option may slow-down the game, and eat up huge memory, so watch out.
OFF No music
WAV Play wav format music
MP3 (NO ID3 TAG) Play mp3 format music (*1)
MIDI Play mid format music (*2)

*1:MP3 music support is "unofficial". Some MP3 may not play, depending to the codec. MP3 with ID3 tag cannot be used.
*2:There is some disk access when music is changed. This is due to YGS2k's limitations that cannot load the multiple MIDI file to RAM. Another limitation is there is no effect on BGM volume setting and BGM fade speed. Once again, this is due to YGS2k's limitations.

BGM FADE SPEED
Select the fade speed of BGM.
NOTE: There is no effect if you are using MIDI music.

SHOW FPS
Select whether to display current FPS (Frames per second).
OFF Disable
ON Enable, display to the game screen
CAPTION Enable, display to the title bar of the window (not visible in full screen mode)

FPS
Select the default FPS (Frames per second).

BACKGROUND
Select whether to display background image.
OFF Disable
ON Enable

HOLD SOUND
Select whether to play se\hold.wav when the Hold feature is used.
INITIAL ONLY Initial Hold only
ON Enable
OFF Disable

LANGUAGE
Select language. This effects to some plugins.
JAPANESE Japanese
ENGLISH English

E BUTTON
Select what E button will do when pressed.
180 ROTATE 180 degree rotation
REVERSE ROTATE Reverse rotation (same as B button)

ENABLE SYSTEM BGM
Select whether to play title screen music (bgm\title.wav) and selection screen music (bgm\select.wav).
ON Enable
OFF Disable

NOTE:Enabling this option will slow-down the start up of the game, and eat up huge memory, so watch out.

EFFECT SPEED
Select the animation speed of line-clear.
AUTO The speed of animation is depending in the current Tetromino's speed.
X 1-5 Always use this speed, regardless of the current Tetromino's speed.

BLOCK SHADOW
Select whether to display "block shadow" (or 3D style objects).
OFF Disable
LEFT Show shadow to left to the blocks
RIGHT Show shadow to right to the blocks

HOLD
Select whether to use Hold feature.
OFF Disable
ON Enable
INPUT SETTING
You can change settings of keyboard or joystick here.

[1P KEYBOARD SETTING]
Set key assignments of player #1.

[2P KEYBOARD SETTING]
Set key assignments of player #2.

[1P JOYSTICK SETTING]
Set joystick button assignments of player #1.

[2P JOYSTICK SETTING]
Set joystick button assignments of player #2.

[JOYSTICK MAX BUTTONS]
Select how many buttons can be used in joystick.

[INPUT TEST]
You can test your settings of keyboard or joystick here. Press A+B to exit.
SCREEN SETTING
You can change the various screen related settings here.

SIZE
Select the screen size and window or full screen mode.

MEMORY
If you have problems on graphics rendering, set this to SYSTEM MEMORY (SLOW).
Otherwise, you should set to VIDEO MEMORY (FAST).

COLOR
Select the color depth of full screen mode. 16 BIT is highly recommended.
In the window mode, this option is ignored. Instead, the game uses Windows current color depth.

USE HIRES
Select whether to use high resolution graphics in 640x480 screen.
OFF Twice the size of normal graphics. (There is a possibility that the screen grows dim in some video cards.)
ON Use high resolution graphics. Enabling this option will slow-down the start up of the game, and eat up huge memory, so watch out.
