プラグインの追加と削除


付属のheboplugin.exeを使用すると、起動時に読み込むプラグインを追加したり、削除したりできます。
ウィンドウにプラグインファイルをドラッグ＆ドロップするか、追加ボタンをクリックするとプラグインを追加することができます。

追加(A)...
プラグインをリストに追加します。Ctrlキーを押しながらファイルをクリックすることで複数選択が可能です。

削除(R)
選択中のプラグインをリストから削除します。（プラグインファイルそのものを削除するわけではありませんのでご安心ください）

上に移動(U)
選択中のプラグインを上に移動します。

下に移動(D)
選択中のプラグインを下に移動します。

OK(O)
変更を保存して終了します。

キャンセル(C)
変更を保存しないで終了します。
