Replay
When the game ends, a screen like the left will appear.
Your play can be saved as a replay file by choosing "SAVE" and pushing A button.
Pushing LEFT or RIGHT button to select the slot of replay file.

Select "RETRY" to restart the game.
Select "END" to return to the title screen.
The replay file you saved can be watched from "REPLAY" in the main menu.
Select the replay file and press A button to play. (Both rule plugin and mode plugin must be loaded!)
You can see some details about the replay file by pushing C button.

Replay is saved below the replay folder in the folder of Heboris.
The replay can be deleted by erasing the .SAV file in this folder.
Controls of replay:
UP Fast-forwarding (x2)
DOWN Fast-forwarding (x4)
A Fast-forwarding (x6)
B Return to the title screen
D This is "Rerecord" feature. It will stop the replay, and the player can play freely.
