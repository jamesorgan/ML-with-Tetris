Special Keys
PAUSE
Press PAUSE button will pause the game. Press PAUSE button again to resume the game.
In the pause screen, while pressing C button, will show the entire playfield.
Press A+B at the same time will quit the current game and return to the title screen.
WARNING: Your game doesn't recorded in the ranking or replay!

GIVEUP
Press this button will quit the current game and return to the title screen.
WARNING: Your game doesn't recorded in the ranking or replay!

SCREEN SHOT
It will save the screen shot into "ss" folder.
The file name of screen shot is ss_0000.bmp, ss_0001.bmp, ss_0002.bmp, and so on.
The file that already exists is overwrited.
WARNING: If you quit the Heboris, the count of screen shot will return to 0000, so make sure to rename the previous screen shot file first!

EXTRA KEY
If the debug mode is enabled in heboris.ini, it will show the debug log. Otherwise, it does nothing.

FPS +1
This increases max FPS (Frames per second) by one. The maximum FPS is 180.

FPS -1
This decreases max FPS (Frames per second) by one. The minimum FPS is 1.
