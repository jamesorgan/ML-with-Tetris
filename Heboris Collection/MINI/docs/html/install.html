インストール方法
新規導入する場合
1. http://yaneurao.hp.infoseek.co.jp/ygs2k/からyaneuraoGameScript2000version1.64cをダウンロードします。
2. ダウンロードしたlzhファイルを解凍すると出てくるYGS2000.exeをヘボリスのフォルダにコピーします。
3. YGS2000.exeをHEBORIS.EXEに名前を変更します。
   （これは任意ですが、説明書ではファイル名がHEBORIS.EXEだと仮定して呼んでいます）
4. HEBORIS.EXEを実行します。

本家に上書きする場合＆旧バージョンからのアップグレードの場合
解凍したファイル全てをヘボリスのフォルダにコピーします。
（上書き確認が出た場合は全て上書きしてください）

Windows XP以降のOSでの注意
Windows XP、Windows Vista、およびそれ以降のWindows OSで正常に動かすには、以下の手順で互換モードを有効にしてください。
１．HEBORIS.EXEを右クリックし、[プロパティ] をクリックします。
２．[互換性] タブの [互換モードでこのプログラムを実行する] をクリックし「Windows 2000」をクリックします。
３．OKボタンをクリックします。

設定を変更する方法
設定変更は全てゲーム内のCONFIG画面で行います。今までセッティングユーティリティで行っていた設定はここでしか行えません。
