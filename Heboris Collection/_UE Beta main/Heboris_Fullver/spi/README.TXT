■このソフトは

  Susie32 で PNG (Portable Network Graphics) 形式の画像ファイルを
  表示させるための Plug-in です。

  この PNG Plug-in は、別に公開している Susie32 PNG Plug-in
  (Regular 版) からガンマ補正などの拡張機能を取り除き、一般的な
  Susie Plug-in としての基本機能だけを残した、軽量版の PNG Plug-in
  です。Regular 版 の拡張機能を全く使用しない場合や、Susie 以外の
  ソフトから利用する場合で Regular 版 の拡張機能が邪魔になる場合
  などにご利用ください。

  この Light 版 PNG Plug-in には、Regular 版 にある以下の機能が
  ありません。

    ・ガンマ補正
    ・背景色の合成
    ・ファイル情報の出力
    ・エラーメッセージの表示

■動作可能な OS

  Win32 のプログラムを動作させることのできる OS (Windows95/98/Me/NT/
  2000/XP など) が必要です。

  当プラグインの以前のバージョンでは、Windows 3.1 + Win32s での動作
  にも正式に対応していましたが、Ver.0.24L 以降、Win32s には非対応に
  なりました。Win32s で動作させる必要がある場合は、Win32s 対応の
  最終バージョンである Ver.0.23L が、一次配布元の最新版を公開している
  場所からダウンロードできますので、そちらをご利用ください。


■インストール

  Susie がインストールされている場合は Susie のあるフォルダに、そう
  でない場合は Windows の System フォルダに、IFPNG.SPI をコピーして
  ください。基本的に、あとは何もする必要はありません。Susie 以外の
  ソフトの場合で、プラグインファイルのインストール場所が指定されて
  いる場合は、その指定に従ってください。


■対応しているファイル形式

  すべての形式の PNG ファイルに対応しています(動作確認済みです)。

  なお、ピクセルの形状が正方形でない PNG ファイルにも対応しています
  が、Susie Plug-in ではピクセルの縦横比を Susie に対して通知する
  だけで、実際の画像の縦横比変更は Susie 側で行います。従って、
  Susie では正常に表示できますが、Susie 以外の Plug-in 対応ソフトで
  正常に表示できるかどうかはそのソフトに依存します。


■設定のしかた

  この Plug-in では、いくつかの機能について、Susie の「設定」→
  「プラグイン」で動作設定を行なうことができます。

  ほとんどの設定項目はインストール直後の初期設定のままでＯＫのはず
  ですが、設定変更をする場合は以下の説明を参考に行なってください。

  ◆「ファイル終端まで内蔵テキストを探す(T)」

    PNG ファイルでは、画像データのあとに内蔵テキストを格納すること
    も可能になっています。この項目を ON にすると、画像データのあと
    にある内蔵テキストも検索するようになります。逆にこの項目を OFF
    にすると、画像データの直前で内蔵テキストの検索を終了します。

    Susie Plug-in では、まずいったん内蔵テキストの検索を行い、
    その後に画像データの読み込みを行うようになっています。よって、
    ファイルの終端まで内蔵テキストの検索をすると、結果的にファイル
    全体を二度読みすることになり、特にフロッピーディスクから直接
    読み込んだ場合には画像の読み込み開始が遅れます。

    実際にはディスクキャッシュが有効に働くため、合計の読み込み時間
    はあまり変わりませんが、画像データの読み込み開始を早めたい場合
    にはこの項目を OFF にしてください。ただし、画像データのあとに
    ある内蔵テキストは全く検知できなくなります。

  ◆「可能ならば破損ファイルも読み込む(K)」

    ファイルが途中で切れているような破損ファイルでも、可能な限り
    読み込んで表示するようにします。

    正確には、画像データ本体を読み込んでいるときにエラーが起こった
    場合で、かつ、それまでに少しでも画像データが展開できているとき、
    その不完全な画像データをそのまま出力します。したがって、ファイル
    ヘッダなど画像データ本体以外の部分を読み込んでいるときにエラー
    が起こった場合や、画像データがまったく展開できていない場合には、
    通常どおりの読み込みエラーになります。

    なお、この項目が OFF の場合でも、画像データが完全に展開された
    あとに起こったエラー(ファイル後部にある内蔵テキストを読んでいる
    ときに起こったエラーなど)はエラー扱いにはならず、正常に画像
    データを出力します。

    インターネット経由で受信されたファイルなどを表示させる機会が
    多い場合には ON にしておくと便利です。なお、この項目を ON に
    した場合は、この Plug-in で画像を読み込めても、他のソフトでは
    破損ファイルエラーが出て画像を読み込めない場合があることを
    覚えておいてください。

■姉妹品の紹介

  ガンマ補正や背景色合成などの拡張機能を装備した Regular 版 の
  PNG Plug-in もあります。PNG では、画像を表示する際にはガンマ補正を
  行なうことが推奨されているため、(画像ファイル変換などではなく)
  Susie で画像表示を主に行なう目的で PNG Plug-in を使用する場合は、
  Regular 版 の方を使用することをお薦めします。

  この Susie32 PNG Plug-in (Regular 版) は、上の一次配布元ホーム
  ページからダウンロードできます。
